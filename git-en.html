Basic commands

A partir d'un repository existant

                    
                        git clone https://github.com/gjambet/maven-course.git

                        git clone git@github.com:gjambet/maven-course.git
                        git clone ./existing-repo ./new-repo

change every thing and ...

                    
                        git commit
                        git push

that's it !

w8 ! commit AND push ?

git dissocie complétement la creation d'un increment de code (git commit) de celle de sa publication (git push)

git pull

Consulter mon historique

                    
                        git log

Etat de ma copie de travail

                    
                        git status

Consulter les modifications 

                    
                        git diff
                        git diff $commit-id1 $commit-id2

Supprimer toute mes modifications depuis le dernier commit ?

                    
                        git reset --hard
