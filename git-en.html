Svn like commands

A partir d'un repository existant

                    
                        git clone https://github.com/gjambet/maven-course.git

                        git clone git@github.com:gjambet/maven-course.git
                        git clone ./existing-repo ./new-repo

change every thing and ...

                    
                        git commit
                        git push

that's it !

w8 ! commit AND push ?

git dissocie complétement la creation d'un increment de code (git commit) de celle de sa publication (git push)

mais alors, je peux créer un repo local ?

                    
                        cd to your-project-root
                        git init
                        git add .

et consulter mon historique ? 

                    
                        git log

et consulter les modifications 

                    
                        git diff
                        git diff $commit-id1 $commit-id2

et supprimer toute mes modifications depuis le dernier commit ?

                    
                        git reset --hard

Et le connecter/déposer à un server distant ?

                    
                        git remote add origin git@github.com:gjambet/your-project.git
                        git push -u origin master

un nouveau svn ... mais pas que !

git pull

git fetch
